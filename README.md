# Soul Code分析学 MVP

「あなただけのSoul Code」を授与する体験を中心にした、静かな高級感のあるWebアプリMVPです。

## 含まれる機能

- トップページ
- 入力フォーム
- Soul Code生成ロジックの仮実装
- Soul Code授与画面
- ギフト結果画面
- Soul Code Passport画面
- Passport画像保存とPDF用印刷表示
- 有料商品案内ページ
- LINE登録導線の仮ボタン
- Meta API投稿テンプレート管理の土台
- 管理画面の土台

## 起動

```bash
npm run dev
```

ブラウザで `http://localhost:4173` を開きます。

## 次に接続できるもの

- Stripeなどの決済
- 公式LINE URL
- Meta APIの投稿実行エンドポイント
- 30〜50ページPDF生成エンジン
- バックエンドDBと管理者認証
